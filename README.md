# Jeu-Bzzz-un-jeu-con-u-en-Python.
Dans ce jeu, quatre joueurs dirigent des abeilles pour récolter du nectar et le ramener à la ruche. Le nectar permet de gagner et de créer de nouvelles unités. Entre récolte, combats et stratégie, ouvrières, bourdons et éclaireuses s’affrontent pour dominer les fleurs. Que le meilleur essaim gagne. Bzzz.
